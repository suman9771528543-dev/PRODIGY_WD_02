## 📋 Project Description
A fully functional, responsive Stopwatch web application built using **HTML5, CSS3, and Vanilla JavaScript**. This project focuses on precise time tracking and smooth user interaction through DOM manipulation.

### Key Features:
* **Start/Pause/Resume:** Seamless control over the timer.
* **Reset:** Clear the timer and all recorded data.
* **Lap Recording:** Capture specific time intervals without stopping the main clock.
* **Responsive Design:** Optimized for both mobile and desktop screens.
* **Modern UI:** Clean, professional interface with high readability.

## 🛠️ Technologies Used
* **HTML5:** For structured content.
* **CSS3:** For styling and responsive layout.
* **JavaScript (ES6):** For core logic, interval timing, and event handling.
